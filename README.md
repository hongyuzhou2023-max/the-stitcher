# 拼图圣手 The Stitcher

纯本地胶片拼图工具：冲扫图导入、裁剪拼版、高画质导出，适合发小红书。图片不离开你的设备。

**V1.0 · by Zhou Hongyu**

## 使用

**在线版：<https://hongyuzhou2023-max.github.io/the-stitcher/>**

或者**直接下载并双击项目根目录的 `index.html`**（约 0.4 MB，自包含单文件），用浏览器打开即可。无需安装、无需联网、图片全程本地处理。

首次打开会选择中文 / English，并进入简短新手指引。

> 若整个文件夹很大：那是开发用的 `node_modules`，可忽略；日常使用只需 `index.html`。

## 开发者

```bash
npm install
npm run build          # 重新生成可双击的 index.html
npm run dev            # 热更新开发
```

进度见 [`project_state.md`](project_state.md)。
